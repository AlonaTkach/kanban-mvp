# Kanban Project

## Business requirements

Build an MVP (a prototype) of a Kanban-style project management application as a web app.

- There should be only **one board**.
- The board has **five fixed columns** that can be **renamed**.
- Each card has a **title** and **details** only.
- A **drag-and-drop** interface to move cards between columns.
- The user can **add a card** and **delete an existing card** — no more functionality than that.
- **No** archive. **No** search or filtering.
- The priority is a **slick, professional, gorgeous UI** with very simple features.
- The app should **open with dummy data** already populated.

## Technical details

- Build it as a **Next.js** app.
- Put it in a subdirectory called **`frontend`**.
- **No persistence** — nothing is saved.
- **No user login** or authentication.
- Use **popular libraries**, keep it as **simple as possible**, but with an **elegant UI**.
